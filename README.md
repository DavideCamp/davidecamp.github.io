# n8n Automation Landing

Sito one-page moderno in React + Tailwind CSS con scena 3D (React Three Fiber) per presentare servizi di automazione n8n.

## Stack

- React + Vite
- Tailwind CSS
- Three.js + @react-three/fiber + @react-three/drei
- Framer Motion

## Avvio locale

```bash
npm install
npm run dev
```

Apri `http://localhost:3000`.

## Build produzione

```bash
npm run build
npm run preview
```
